Overview

APNs 의 remote notification payload 에서 특정 리소스를 다운로드 받아 해당 리소스를 볼 수 있도록 처리하는 클래스

앱에서 HNPushMgr 클래스와 notification service app extension 을 동시에 사용할 경우,

이 클래스를 확장해야만 이미지, 동영상 등의 리소스가 정상적으로 처리될 수 있음.

테스트를 위해서는 아래의 샘플을 이용할 것

{ “aps”: { “alert”: { “title”: “Hi,There~”, “body”: “Do you see it?”, “imageURL”: “http://img.hunet.co.kr/Files/mall/imagine/171012131056.jpg or http://www.ithinknext.com/mydata/board/files/F201308021823010.mp4”, “version”: “integerValue” }, “sound”: “default”, “badge”: 1, “category”: “앱의 번들 아이디”, “mutable-content”: 1, } }
