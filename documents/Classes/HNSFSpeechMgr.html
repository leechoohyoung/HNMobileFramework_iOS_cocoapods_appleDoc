Overview

단말기의 마이크를 이용하여 사용자의 음성 데이터를 녹음하고 녹음된 음성 데이터에 대한 문자열 추출을 담당하는 클래스

이 클래스가 정상적으로 동작하기 위해서는 앱의 Info.plist 에 NSSpeechRecognitionUsageDescription, NSMicrophoneUsageDescription 키와 키에 대한 안내 문자열이 정의되어야 함.

또한, 이 클래스는 iOS 10.0 부터 지원됨.

– startListen

사용자의 음성 데이터 분석 시작

- (BOOL)startListen

Return Value

AVAudioEngine 이 성공적으로 시작했다면 YES, 그렇지 않으면 NO

Discussion

이 메소드가 호출되기 전에 반드시 [HNSFSpeechMgr prepareToListen:originVoiceFileURL:compressedVoiceFileURL:locale:] 메소드를 한 번은 호출해야 함.

또한, 분석을 종료하려면 [HNSFSpeechMgr stopListen] 메소드를 호출해야 함.

음성 인식을 위해 주어지는 시간은 1분 정도이므로 UI 에 적절한 표시가 필요함.
