– stringByRemovingURLProtectionCharacter

[] 이 제거된 문자열 리턴

- (instancetype)stringByRemovingURLProtectionCharacter

Return Value

[] 이 제거된 문자열

Discussion

UIWebView 를 이용하여 커스텀 프로토콜을 사용할 때 URL 값을 전달받아야 하는 경우가 있는데, 이 때, URL에 2개 이상의 매개변수가 포함되면 해당 정보들이 누락되는 경우가 발생하여 [] 붙여서 처리하고 있음. 예를 들어, popup://?pageToOpen=http://m.happyceo.co.kr?param1&param2 형식의 정보를 수신하게 되면 pageToOpen 값은 http://m.happyceo.co.kr?param1&param2 으로 설정되어야 하지만 실제로는 http://m.happyceo.co.kr?param1 으로 설정되므로 pageToOpen 값이 변경되지 않도록 popup://?pageToOpen=[http://m.happyceo.co.kr?param1&param2] 로 전달하고 있음. 자세한 문서는 링크 참고 : https://docs.google.com/spreadsheets/d/1lKpdN4C8ooiJy-_J3wEFBCevQ0bXD-raFgwTPMMwvb4/edit#gid=0
