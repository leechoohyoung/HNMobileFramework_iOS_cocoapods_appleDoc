Overview

Remote Notification 등록 결과로 전달되는 device token 변환을 위한 카테고리

– hexStringDeviceToken

NSData 타입의 device token 을 NSString 타입으로 변환

- (NSString *)hexStringDeviceToken

Return Value

NSString 타입의 device token

Discussion

메소드 내부에서 비트 연산(& 0xff)을 이용하여 하위 8bit 추출을 진행함.

예를 들어, NSData 를 byte 로 복사한 결과가 “\x2d” 일 경우, NSString 으로 변환하면 ffff2d 가 됨.

0xffff2d 는 11111111 11111111 11111111 11111111 00101101 이고

0x0000ff 는 00000000 00000000 00000000 00000000 11111111 이므로,

두 값을 & 연산하게 되면 00000000 00000000 00000000 00000000 00101101 값을 얻게 되는데,

이는 결과적으로 하위 8 비트만을 추출한 것과 동일함.
